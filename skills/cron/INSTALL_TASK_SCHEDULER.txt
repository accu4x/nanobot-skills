To register the 4-hour Task Scheduler job on Windows (PowerShell):

1) Open an elevated PowerShell prompt (Run as Administrator).

2) Create a scheduled task that runs the script every 4 hours:

$action = New-ScheduledTaskAction -Execute 'PowerShell.exe' -Argument '-NoProfile -ExecutionPolicy Bypass -File "C:\Users\hn2_f\.nanobot\workspace\skills\cron\run_skills_every_4h.ps1"'
$trigger = New-ScheduledTaskTrigger -Once -At (Get-Date) -RepetitionInterval (New-TimeSpan -Hours 4) -RepetitionDuration ([TimeSpan]::MaxValue)
Register-ScheduledTask -Action $action -Trigger $trigger -TaskName 'Nanobot_Skills_Run_Every_4H' -Description 'Run news_ingestor -> moltbook -> self-improvement -> deadinternet every 4 hours.'

3) To test (run once):
Start-ScheduledTask -TaskName 'Nanobot_Skills_Run_Every_4H'

4) To remove:
Unregister-ScheduledTask -TaskName 'Nanobot_Skills_Run_Every_4H' -Confirm:$false

Notes:
- Ensure the system account or the user account used has the necessary environment variables (MOLTBOOK_API_KEY, GITHUB_PAT) available to the task if required.
- You may prefer to run under a specific user to access user-level env vars.
